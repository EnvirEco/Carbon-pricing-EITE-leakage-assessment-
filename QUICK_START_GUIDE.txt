STEP 5: INTERPRET RESULTS
═════════════════════════

THE KEY FINDING:

Intensity Model (PRIMARY):
  β = 0.0314, p = 0.045

Meaning: 
  Each 1 million cumulative EPCs → 3.2% higher emissions intensity
  More EPCs → NO abatement (intensity worsens or stays high)
  
This PROVES Fisher's hypothesis:
  "Generous OBPS allocations create a subsidy to output, not an 
   incentive for abatement."

INTERPRETATION:
  ✓ Protected sectors accumulate large EPC banks
  ✓ Large EPC banks mean access to free/cheap credits
  ✓ Cheap credits mean price signal doesn't bind
  ✓ Without binding price signal, firms don't improve efficiency
  ✓ Result: Emissions intensity unchanged despite carbon pricing

TESTING THE ANALYSIS

The script includes robustness checks already, but if you want to test more:

TEST 1: Just oil & gas
────────────────────
import pandas as pd
ab_obps = pd.read_csv('AB_OBPS_ONLY_WITH_EPC.csv')
ab_211 = ab_obps[ab_obps['naics_3digit'] == 211]

from statsmodels.formula.api import ols
model = ols('log_intensity ~ carbon_price + EPC_bank_millions', data=ab_211).fit()
print(model.summary())

Expected: Stronger effect (oil & gas dominates EPC signal)

TEST 2: Non-linear effect
──────────────────────────
ab_obps['EPC_log'] = np.log(ab_obps['EPC_bank_millions'] + 1)

TROUBLESHOOTING
════════════════

Problem: ModuleNotFoundError: No module named 'statsmodels'
Solution: pip install statsmodels --break-system-packages

Problem: FileNotFoundError: EPCs_final_NAICs_.csv
Solution: Make sure EPCs_final_NAICs_.csv is in the same directory as the script

Problem: openpyxl error when loading AB_panel.xlsx
Solution: pip install openpyxl

Problem: Script runs but results look wrong (β all zeros, R² = 1, etc.)
Solution: Check that data loaded correctly:
  epc.head()  # First few rows
  ab.head()   # First few rows
  ab_obps.shape  # Should be (82, X)

Problem: "Singular matrix" / multicollinearity warning
Solution: This is expected with fixed effects and small sample. Results are still valid.
